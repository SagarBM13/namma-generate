## 1. Introduction

A generator to generate mvp-rockets projects/boilerplate codes.

## 2. Pre-requirement

- Ubuntu 20.04.4 LTS
- docker(19.xx)
- docker-compose(1.28.xx)

## 3. Installation

```
npm i -g @mvp-rockets/namma-generator

```

## 4. Create the project/boilerplate codes.

```
namma_generate
```
